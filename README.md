# MamPokoj - Rental Room Discovery Platform

MamPokoj is a web platform for discovering and managing rental rooms. It allows users to browse listings, manage their own rooms, and interact with a simple dashboard experience optimized for both desktop and mobile.

## Features

- Browse available rental rooms
- View detailed room listings
- User dashboard for managing listings
- Add and delete room images
- Responsive design for mobile and desktop use
